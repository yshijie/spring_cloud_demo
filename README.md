# spring_cloud_demo
spring_cloud demo project

该工程为SpringCloud框架学习使用，集成SpringCloud中Eureka、Zuul、Ribbon、Hystrix等组件；

### 相关组件

注册中心--Eureka  [GitPress](https://gitpress.io/@yangshijie/spring_cloud_eureka) 、 [简书](https://www.jianshu.com/p/ec8f5b8da5cf)

动态路由--Zuul

负载均衡--Ribbon

熔断器--Hystrix

服务调用--Feign
